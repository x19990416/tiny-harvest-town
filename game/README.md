# Game Client｜Godot 客户端

此目录用于放置 Tiny Harvest Town 的 Godot 4.x 游戏客户端。

## 第一阶段客户端目标

- 小地图加载。
- 玩家移动。
- NPC 对话。
- 读取 JSON 内容。
- 触发任务。
- 展示数学题。
- 校验答案。
- 发放奖励。
- 推进建筑修复进度。
- 本地保存学习与游戏进度。

## 推荐模块

- ContentLoader
- TaskSystem
- ExerciseSystem
- InventorySystem
- DialogueSystem
- BuildingProgressSystem
- SaveSystem

当前文件只是占位项目，用于后续 Godot 原型开发。
