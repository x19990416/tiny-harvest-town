# Tiny Harvest Town｜小禾镇

**Tiny Harvest Town（小禾镇）** 是一款面向儿童的知识成长型像素生活游戏。孩子在种田、建造、交易、探索和认识村民的过程中完成数学、语文、英语等轻量练习，让知识变成小镇成长的力量。

第一阶段不做全年级、全学科、开放大世界，而是先做一个可验证 MVP：

**Tiny Harvest Town: Grade 3 Math Farm Season｜小禾镇：三年级数学农场季**

核心验证目标：孩子是否愿意为了让小镇恢复生机而主动完成练习；家长是否能理解其学习价值并接受后续成长包/主题季模式。

## 一句话定位

做一点练习，种一座小镇。

## 产品原则

1. 不把题目硬塞进游戏，而是让练习成为小镇生活的生产关系。
2. 孩子看到的是农场、村民、订单、建筑修复和四季变化；家长看到的是知识点、熟练度、错题和成长报告。
3. 错题不做惩罚式失败，而转化为温和反馈和再次尝试。
4. 第一版只做“三年级数学 + 农场经营 + 小镇修复”，先验证核心循环。
5. 内容必须配置化，后续年级包、学科包、主题季不能依赖频繁改代码。

## MVP 范围

首发 MVP 只包含四个核心场景：

- 农场：种植、浇水、收获，对应基础计算、倍数关系、数量规划。
- 商店：买种子、卖作物、找零钱，对应加减乘除、价格计算。
- 工坊：制作面包、果酱、木板，对应单位换算、配方、平均分。
- 村民订单：完成配送任务，对应三年级应用题。

首发主线：**修复面包店**。

## 技术建议

- 游戏客户端：Godot 4.x
- 内容配置：JSON
- 后台管理：Vue / React + Spring Boot 或 FastAPI
- 数据库：PostgreSQL 或 MySQL
- 美术：Aseprite / Photoshop / Pixelorama
- 版本管理：GitHub
