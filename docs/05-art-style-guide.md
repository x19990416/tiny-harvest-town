# 05｜像素美术风格指南

## 总体气质

Tiny Harvest Town 的画面应当是低饱和、温暖、清爽、儿童友好的田园像素风。

它可以参考生活模拟游戏的亲和力，但不应完全复制任何既有游戏。视觉要比成人独立游戏更明亮、更清楚、更低压力。

## 核心关键词

温暖、克制、清晰、圆润、田园、成长、四季、小镇、低压力。

## 建议规格

地图格子：16px 或 32px。MVP 建议 32px，便于儿童识别。  
角色基础尺寸：32×32 或 32×48。  
道具尺寸：16×16 / 32×32。  
UI 图标：32×32 起。  
对话框字号：优先可读性，不为了像素感牺牲阅读。

## 视角

建议使用正交俯视或轻度伪俯视，不建议复杂等距视角。原因是儿童产品需要交互清楚，农田格子、商店、工坊、路径都要容易理解。

## 色彩

使用低饱和暖色作为基础，避免过度霓虹和高对比。不同功能区域要有明确色彩区分：

- 农场：浅绿、麦黄、泥土色。
- 商店：暖橙、木色。
- 工坊：木色、石灰色、浅蓝灰。
- 面包店：奶油色、烘焙黄、浅棕。
- 知识反馈：柔和金光、种子发芽、灯光亮起。

## 角色设计

角色轮廓要简单，有明显识别点。不要让角色过度成人化或过度幼儿园化。

建议角色：

- 禾禾：小镇引导角色，可以有禾苗、种子、灯笼或小精灵特征。
- 米粒：商店老板，圆润、亲切、围裙。
- 阿木：工坊老板，木匠气质，工具包。
- 村长：温和、稳定，有旧钥匙或小镇地图。
- 面包店老板：温暖、有面包帽/围裙元素。

## 建筑修复视觉

建筑修复是核心反馈，必须做出阶段变化。

面包店阶段：

1. 破旧状态：招牌歪斜、窗户灰暗。
2. 初修状态：门修好、灯亮起。
3. 可运营状态：炉子点火、窗边有面包。
4. 完成状态：招牌完整、村民排队、香气动画。

## UI 原则

题目区域必须明确，不要被地图元素干扰。  
按钮要大，点击区域要宽。  
正确反馈使用温和动效。  
错误反馈不闪红、不震动吓人。  
家长端可以更简洁，但孩子端必须更有故事感。

## 动画优先级

MVP 必做：

- 玩家走路。
- 播种、浇水、收获。
- 建筑修复变化。
- 材料飞入背包。
- 正确反馈的轻微发光/发芽。
- NPC 对话气泡。

可后置：

- 四季完整替换。
- 天气系统。
- 宠物复杂动画。
- 大量村民日程动画。
